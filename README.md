# Contenido clases PP1 JS

Este repositorio tiene todo el contenido de ejemplo trabajado en las clases.   


## Estructura del repositorio
El contenido de cada clase está separado por ramas.


## Cómo traer el repositorio a tu máquina local

Seguí estos pasos para trabajar con los ejemplos en tu compu:

### Clonar el repositorio

(Tenes que tener git instalado en la maquina)

1 Abrí tu terminal y ejecuta:

```bash
git clone https://github.com/valentinoarballo/contenido-clases-segundo.git
```

2 Entra al directorio del proyecto

```bash
cd contenido-clases-segundo
```

3 Cambia a la rama de la clase que quieras

Por ejemplo, para acceder al ejemplo de la Clase 1:
```bash
git checkout clase1
```

### Cómo ver un ejemplo de clase
Cambia a la rama correspondiente usando tu cliente Git:
```bash
git checkout clase1
```

Ahora tu carpeta local tiene todos los archivos de esa clase.
